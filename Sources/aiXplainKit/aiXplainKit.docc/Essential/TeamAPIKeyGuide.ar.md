# دليل مفتاح API للفريق

تعلّم كيفية إنشاء مفاتيح API واستخدامها.

## كيفية الحصول على مفاتيحك

 [أنشئ حسابًا](https://platform.aixplain.com/register) أو [سجّل الدخول](https://platform.aixplain.com/login) على منصة aiXplain. ثم من لوحة التحكم، انتقل إلى صفحة [التكاملات](https://platform.aixplain.com/account/integrations).

![صورة لوحة التحكم](NavigateAPIKey)

### إنشاء مفتاح API جديد
في صفحة **التكاملات**، يمكنك العثور على زر **إنشاء مفتاح وصول للفريق** في الزاوية العلوية اليمنى. يمكنك إنشاء مفتاح جديد بالنقر على هذا الزر، ثم تحديد تسمية وتاريخ انتهاء صلاحية (اختياري).

### إدارة مفاتيح API
في صفحة **التكاملات**، يمكنك عرض جميع مفاتيح API الخاصة بالفريق الموجودة. كما يمكنك حذف المفاتيح من هذه الصفحة.

### ضبط المفاتيح

مثال على كيفية استخدام `APIKeyManager` لاسترجاع مفاتيح API وضبطها.

لضبط مفاتيح API باستخدام متغيرات بيئة Xcode، اتبع الخطوات التالية:

1. في Xcode، حدد مشروعك في متصفح المشروع.
2. حدد الهدف الخاص بك، ثم انقر على علامة التبويب "Info".
3. ضمن قسم "Configurations"، انقر على زر "+" في الزاوية السفلية اليسرى.
4. في الصف المُضاف حديثًا، عيّن "Name" إلى اسم مفتاح API المطلوب (مثلاً "TEAM_API_KEY") و"Value" إلى مفتاح API الخاص بك.
5. كرر الخطوة 4 لكل مفتاح API تحتاج إلى ضبطه.

بعد ضبط متغيرات البيئة، سيقوم `APIKeyManager` تلقائيًا بتحميل مفاتيح API واستخدامها من متغيرات البيئة المقابلة.

يمكنك أيضًا ضبط مفاتيح API مباشرةً في الكود عند الحاجة:

```swift
AiXplainKit.shared.keyManager.TEAM_API_KEY = "<Your Key>"
```
